# Contributing

학습 공개 리포지토리. PR/Issue 환영 — 커리큘럼 개선 아이디어, 오타 수정, 실습 코드 버그, 새 자료 제보.

## 방식

- **Issue 먼저**: 큰 변경은 먼저 issue로 논의.
- **작은 수정**: 오타/링크 깨짐은 바로 PR.
- **새 자료 제보**: `resources/links-classified.md` 양식 따라서.
- **커리큘럼 변경**: 주차별 문서와 `curriculum/00-roadmap.md` 함께 업데이트.

## PR 기준

- `.env.example` 외에 절대 시크릿 포함 X
- 스타일: ruff default
- 이모지 추가 지양 (docs 포함)
- 한국어/영어 혼용 OK. 일관성만 지켜주세요.

## 학습자 피드백

이 커리큘럼을 실제로 진행해보고 뒤처진 곳/과했던 곳 피드백을 issue로 주시면 후속 학습자들에게 큰 도움이 됩니다.
